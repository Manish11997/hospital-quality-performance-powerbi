# Hospital Quality Performance Dashboard

## Project Overview
This Power BI dashboard provides an interactive analysis of hospital quality performance across the United States. The project evaluates hospital ratings, patient safety measures, readmission performance, birthing-friendly status, and geographic trends.

## Key Features

### Overview Dashboard
- Total Hospitals KPI
- High-Performing Hospitals %
- Average Better Performance %
- Birthing-Friendly Hospitals %

### Geographic Analysis
- State-level performance map
- Top and Bottom performing states
- Average rating comparison

### Performance Measure Details
- Better / No Different / Worse analysis
- Hospital-level matrix
- Drill-through hospital detail page

### Scenario Insights
- What-If parameter analysis
- Safety improvement simulation
- Readmission improvement simulation
- Projected high-performing hospitals
